# Clippy

A cheerful animated paperclip pet inspired by Microsoft’s retired Office Assistant.

Clippy is back as a modern Codex companion, with expressive eyes, flexible paperclip movement, directional gazes, and a complete set of helpful reactions.

> This is a fan-made homage and is not affiliated with or endorsed by Microsoft.

## Features

Clippy includes:

- Idle breathing and blinking
- Running right and left
- Waving
- Jumping
- Failure reactions
- Waiting for approval or input
- Focused working/thinking
- Reviewing
- 16 directional look poses
- Consistent expressive physical eyes and eyebrows
- Codex-compatible v2 sprite atlas

## Origins

Clippy was Microsoft Office’s animated assistant, introduced to help users while they worked on documents. His familiar question—“It looks like you’re writing a letter”—made him one of the most recognizable software characters of the 1990s and early 2000s.

Although Clippy was eventually retired, the character remained a memorable part of computing history. This project resurrects that spirit as a small, expressive companion for modern AI-assisted workflows.

The design keeps the classic paperclip silhouette, oversized eyes, dark eyebrows, and helpful personality while adapting the character for a modern animated-pet format.
